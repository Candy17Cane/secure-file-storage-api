## Русский

# 🔐 Secure File Storage API

API для безопасного хранения файлов с шифрованием.  
Реализовано на FastAPI с авторизацией и защитой данных.

---

## 🚀 Возможности

- 🔑 Регистрация и авторизация пользователей (JWT)
- 📤 Загрузка файлов
- 📥 Скачивание файлов
- 🔐 Шифрование файлов перед сохранением
- 🧂 Использование salt и nonce для безопасности
- 🗂 Хранение файлов на сервере

---

## 🛠 Технологии

- Python 3.12
- FastAPI
- SQLite
- Cryptography
- Uvicorn

---

## ⚙️ Запуск проекта

```bash
git clone https://github.com/Candy17Cane/secure-file-storage-api.git
cd secure-file-storage-api

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

uvicorn app.main:app --reload
